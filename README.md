# Mimiq

Avatar mimic/q extension for Habbo using G-Earth.

## Features

- Copy target user figure and motto
- Mimic actions, dance, signs, and effects
- Follow target movements
- Copy chat messages (talk, shout, whisper)
- Copy typing status etc...

## Requirements

- .NET 8.0 Runtime
- G-Earth
